# Sprint 00 — Configuração Inicial do Projeto

## Objetivo

Preparar toda a estrutura inicial do projeto, configurando o ambiente de desenvolvimento, dependências, organização das pastas e padronização do código.

Esta sprint NÃO deve implementar nenhuma funcionalidade de negócio.

---

# Contexto

Antes de iniciar esta Sprint leia obrigatoriamente:

- agents/ARCHITECT.md
- agents/PROJECT_MANAGER.md
- agents/SENIOR_DEVELOPER.md
- agents/CODE_STYLE.md

Leia também toda documentação da pasta docs/.

---

# Escopo

Esta Sprint deve realizar:

- Configuração do projeto Next.js
- Organização das pastas
- Configuração do TypeScript
- Configuração do Tailwind
- Instalação das dependências
- Configuração do ESLint
- Configuração do Prettier
- Configuração do alias @/\*
- Configuração do MongoDB
- Configuração das variáveis de ambiente
- Estrutura inicial do projeto

Não implementar nenhuma tela.

Não criar APIs.

Não criar Models.

Não criar autenticação.

---

# Critérios de Aceite

Ao finalizar esta Sprint o projeto deve:

- iniciar normalmente
- compilar sem erros
- passar no TypeScript
- passar no ESLint

Toda estrutura de pastas deve estar criada.
